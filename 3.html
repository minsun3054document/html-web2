img, src, 크기조정 width

- img
: 어떤 이미지를 불러올 지에 대한 정보가 부족하므로 속성(attribute) 필요, 도입
- src : source >>이게 바로 속성
```html : code :
img src="https://s3-ap-northeast-2.amazonaws.com/opentutorials-user-file/module/3135/7648.png"
```

속성 값 : 이미지의 주소
>> src라는 속성을 통해 이미지를 부를 수 있게 된 것

인터넷에 있는 링크 주소를 써도 되고
원하는 위치에 이미지 저장해서 컴퓨터 이미지를 표시해도 된다.

웹페이지 파일과 같은 위치에 이미지파일이 있으면
scr속성 값으로 이미지파일 이름만 적어주면 된다.

```html : code :
img src="파일명.jpg"
```

- width : 숫자, % 사용해서 원하는 크기로 사진 크기 조정

```html : code :
img src="https://s3-ap-northeast-2.amazonaws.com/opentutorials-user-file/module/3135/7648.png" width="100%"
```

- 저작권에 구속받지 않고 자유롭게 이미지 사용 가능한 서비스 :
public domain image 검색
혹은 unsplash.com
